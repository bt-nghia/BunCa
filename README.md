# parameters need to be optimized

- `sw`: self weight in asym matrix
- `nw`: neighbor weight in asym matrix (`sw` and `nw` in [0 ; 1])
- `w3`, `w4`: weight while fusing feature in item view (`w3`, `w4` in [0 ; 1])
